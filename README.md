# README

## アプリ概要

「 男女問わず、ゴルフを通じてマッチングできるサービス「GoRo」 」

18歳以上のゴルファーを対象にゴルフ初心者、経験者問わず、マッチングできるサービスです。

男性同士、女性同士でもマッチングができる機能を新たに追加したのが、GoRoのサービスになります。


## 開発言語

* Ruby 3.0.1
* Rails 6.0.3


## 就業Termで学んだ技術
* devise
* Ajax


## カリキュラム外の技術
* ransack


## 実行手順
```
$ git clone git@github.com:masuda-yusei/GoRo-app.git
$ cd GoRo-app
$ bundle install
$ rails db:create db:migrate
$ rails s
```
